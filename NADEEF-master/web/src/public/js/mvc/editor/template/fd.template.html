Functional Dependency (FD) is a constraint between two sets of attributes in a relation. It is composed of left hand and right hand. It means for the equal left hand value tuple the right hand value should also be the same. The syntax of FD Rule is as following

Column A Column B... | Column C Column D...

For example, rule zipcode | city, state indicates that tuples with the same zipcode value should also have the same attribute value on city and state.
